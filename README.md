# Timedctl

Timed command line tool

## Installation

```bash
yay -S elixir
```

## Building
```bash
mix escript.build
```

and published on [HexDocs](https://hexdocs.pm). Once published, the docs can
be found at [https://hexdocs.pm/timedctl](https://hexdocs.pm/timedctl).
